준비하실 항목은 총 3가지입니다.
① 결제 서비스(토스페이먼츠) API 키 발급  ②  문자 발송(NAVER SENS) API 키 발급  ③  인터넷 주소(도메인) 구매
1
결제 서비스 API 키 발급 (토스페이먼츠)
고객이 앱에서 카드·계좌이체로 결제할 수 있게 해주는 서비스입니다
사전 준비물
사업자등록증, 정산받을 계좌번호(사업용), 대표자 신분증 사본
1
토스페이먼츠 개발자센터 회원가입

아래 주소를 인터넷 주소창에 입력하고 접속합니다.

https://developers.tosspayments.com
  1. 오른쪽 위 "로그인" → "회원가입" 클릭
  2. 이메일 주소, 비밀번호를 입력하고 가입 완료
  3. 가입한 이메일로 인증 메일 도착 → 메일의 링크 클릭해서 인증
2
사업자 정보 등록 (실결제 수신을 위해 필수)
  1. 로그인 후 오른쪽 위 이름 클릭 → "내 계정" 또는 "상점 관리" 선택
  2. "사업자 정보 등록" 버튼 클릭
  3. 아래 정보 입력:
    입력 항목내용
    상호명사업자등록증의 상호명 그대로
    사업자등록번호000-00-00000 형식
    대표자명사업자등록증의 대표자 이름
    업태 / 업종사업자등록증 기준
    사업장 주소사업자등록증의 주소
    정산 계좌매출금이 입금될 사업용 계좌
  4. 저장 후 심사 대기 (보통 1~3 영업일 소요)
심사 전에도 "테스트 키"로 기능 확인은 가능합니다. 실제 고객 결제는 심사 승인 후 "라이브 키"를 개발자에게 전달하면 됩니다.
3
API 키 확인 및 복사

로그인 후 아래 주소로 이동합니다.

https://developers.tosspayments.com/my/api-keys

화면에 두 종류의 키가 보입니다:

키 종류시작 문자설명
테스트 클라이언트 키test_ck_...테스트 전용 (돈 안 빠짐)
테스트 시크릿 키test_sk_...테스트 전용 (돈 안 빠짐)
라이브 클라이언트 키live_ck_...실제 결제용 (심사 후 발급)
라이브 시크릿 키live_sk_...실제 결제용 (심사 후 발급)
  1. 각 키 옆의 "복사" 버튼을 클릭합니다
  2. 복사한 값을 메모장에 붙여넣고 저장해 둡니다
시크릿 키는 절대 타인에게 공개하지 마세요. 카드 비밀번호처럼 안전하게 보관하세요.

개발자에게 전달할 정보 (토스페이먼츠)

항목예시 형태
클라이언트 키live_ck_XXXXXXXXXXXX
시크릿 키live_sk_XXXXXXXXXXXX

위 2가지를 이메일 또는 카카오톡으로 개발자에게 전달해 주세요.
전달 후 개발자가 서버에 등록하고 배포합니다.

2
문자 발송 서비스 API 키 발급 (NAVER SENS)
고객 휴대폰으로 인증번호 문자를 보내는 서비스입니다 (이 설정 없으면 로그인 불가)
사전 준비물
네이버 계정 (없으면 새로 가입), 결제 수단 (신용카드 또는 계좌), 문자를 보낼 휴대폰 번호 또는 사업용 전화번호
1
네이버 클라우드 플랫폼 가입
https://www.ncloud.com
  1. 오른쪽 위 "회원가입" 클릭
  2. "개인 사업자" 선택
  3. 이름, 이메일, 휴대폰 번호 입력 후 인증 완료
  4. 결제 수단 등록 (신용카드 또는 계좌이체)
    ※ 문자 1건당 약 8~10원 정도 청구됩니다. 소량이면 월 몇천원 수준.
2
SENS 서비스 신청 및 프로젝트 생성
  1. 로그인 후 상단 메뉴 "Products & Services" 클릭
  2. 검색창에 "SENS" 입력 후 선택
  3. "이용 신청" 버튼 클릭
  4. 왼쪽 메뉴에서 "SMS" 선택
  5. "서비스 신청" 클릭 → 프로젝트 이름에 "세팡" 입력 → 생성
  6. 생성된 프로젝트 선택 → 서비스 ID 값을 복사해서 메모장에 저장
서비스 ID는 ncp:sms:kr:숫자:프로젝트명 형태로 생겼습니다.
3
발신번호 등록 (문자를 보내는 번호)

고객에게 발송되는 인증번호 문자의 발신자 번호입니다.

  1. SENS 콘솔 왼쪽 메뉴 "발신번호 관리" 클릭
  2. "발신번호 등록" 클릭
  3. 번호 유형 선택:
    유형방법소요 시간
    휴대폰 번호본인 명의 휴대폰 → 문자 인증즉시
    유선(대표) 번호통신사 가입 증명서 제출1~2 영업일
  4. 인증 완료 후 등록된 번호를 메모장에 저장
번호는 하이픈(-) 없이 숫자만 저장하세요.
예) 010-1234-5678 (X) → 01012345678 (O)
4
Access Key / Secret Key 확인
  1. 오른쪽 위 이름 클릭 → "마이페이지" 선택
  2. 왼쪽 메뉴 "인증키 관리" 클릭
  3. Access Key ID를 복사해서 메모장에 저장
  4. "Secret Key 보기"를 클릭하여 Secret Key도 복사해서 저장
Secret Key는 처음 생성할 때만 볼 수 있습니다.
반드시 바로 복사해서 안전한 곳에 저장하세요. 분실 시 재발급 필요.

개발자에게 전달할 정보 (NAVER SENS)

항목어디서 확인
서비스 IDncp:sms:kr:... — SENS 프로젝트 정보
Access Key ID마이페이지 → 인증키 관리
Secret Key마이페이지 → 인증키 관리
발신번호등록한 번호 (하이픈 없이, 예: 01012345678)

위 4가지를 이메일 또는 카카오톡으로 개발자에게 전달해 주세요.

3
인터넷 주소(도메인) 구매
sepang.kr 같은 전용 주소로 서비스를 운영하기 위해 필요합니다
도메인이란?
"www.naver.com"처럼 인터넷에서 서비스를 찾을 수 있는 고유 주소입니다. 현재는 임시 주소(vercel.app)로 운영 중이며, 도메인을 구매하면 원하는 주소로 변경됩니다. 도메인은 매년 갱신이 필요합니다.
1
사용할 도메인 결정

원하는 주소를 정하세요. 예시:

예시 도메인연간 비용(가비아 기준)비고
sepang.kr약 22,000원/년한국 도메인, 신뢰감 높음
sepang.com약 18,000원/년국제 도메인, 가장 일반적
sepang.co.kr약 22,000원/년한국 기업용
도메인 1개만 구매하면 됩니다. 나머지 주소(partner., admin., api.)는 개발자가 설정합니다.
2
가비아에서 도메인 구매
https://www.gabia.com
  1. 위 주소에 접속합니다
  2. 메인 화면 검색창에 원하는 도메인을 입력합니다
    예) sepang 입력 후 검색
  3. 사용 가능한 도메인 목록에서 원하는 것 선택 (초록색 "등록 가능" 표시된 것)
  4. "장바구니 담기" → "주문하기" 클릭
  5. 회원가입 또는 로그인 후 결제 진행 (신용카드 또는 계좌이체)
  6. 구매 완료 확인 이메일 수신
구매 즉시 도메인 사용권이 생기며, 개발자가 연결 작업을 진행합니다.
3
가비아 계정 정보 개발자에게 전달

도메인 연결(DNS 설정)을 위해 아래 정보를 개발자에게 전달해 주세요.

전달 방법 선택 (둘 중 하나):

방법 A — 계정 공유: 가비아 로그인 아이디와 비밀번호를 개발자에게 전달
(개발자가 직접 DNS 설정 진행 후, 작업 완료 시 비밀번호 변경 권장)

방법 B — 네임서버 변경: 가비아에서 직접 네임서버를 변경
(개발자에게 "네임서버 변경 방법" 안내 요청 후 직접 진행)

개발자에게 전달할 정보 (도메인)

항목내용
구매한 도메인예) sepang.kr
가비아 아이디가입 시 사용한 아이디
가비아 비밀번호임시 비밀번호로 변경 후 전달 권장

개발자가 DNS 설정 완료 후 24~48시간 이내에 주소가 정상 연결됩니다.
